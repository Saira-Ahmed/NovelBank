The Novel Bank Management System is a console-based application developed using MASM x86 (32-bit Assembly Language) with the Irvine32 library.
This project simulates a basic novel lending system where users can add novels, view available novels, borrow novels, and return them.

It is designed as a lab final / academic project to demonstrate fundamental assembly language programming concepts.
